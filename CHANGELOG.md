# Changelog

All notable changes to `eloquent-super` will be documented in this file.

## 1.3.0 - 2024-03-14

### Added

- PHP 8.3 support
- Laravel 11 support

### Removed

- PHP 8.2 support
- Laravel 10 support

## 1.2.0 - 2023-04-28

### Added

- PHP 8.2 support
- Laravel 10 support

### Removed

- PHP 8.1 support
- Laravel 9 support

## 1.1.0 - 2022-02-11

### Added

- PHP 8.1 support
- Laravel 9 support
- The ability to override the default morph name of the `super` relationship

### Removed

- PHP 8.0 support
- Laravel 8 support

## 1.0.0 - 2021-09-11

- Initial release
